# chart

| 属性         | 说明             | 类型                 | 默认值   |
| :----------- | :--------------- | :------------------- | :------- |
| id           | 元素节点位置     | string               |          |
| width        | 宽度             | number               |          |
| height       | 高度             | number               |          |
| marginLeft   | 左间距           | number               |          |
| marginBottom | 下间距           | number               |          |
| xArr         | 水平方向节点坐标 | number[]             |          |
| yArr         | 垂直方向节点坐标 | number[]             |          |
| valueArr     | 值坐标           | number[]             |          |
| shape        | 下间距           | 'normal' \| 'smooth' | 'normal' |
| mode         | 下间距           | 'normal' \| 'fill'   | 'normal' |
